Penghapusan Akun – Leveling App

Jika Anda ingin menghapus akun dan semua data Anda di aplikasi Leveling App, silakan ikuti langkah-langkah berikut di dalam aplikasi:

  • Buka aplikasi Leveling App
  • Masuk ke Edit Mode (icon gear) > Edit Profile > Profile Detail > Hapus Akun
  • Ikuti instruksi untuk mengonfirmasi penghapusan akun

Data yang Dihapus

Setelah dikonfirmasi, sistem kami akan segera dan secara permanen menghapus data berikut:

  • Data akun pengguna (termasuk email dan ID pengguna)
  • Data kemajuan seperti quest, XP, skill, dan item

Kami tidak menyimpan data Anda setelah proses penghapusan selesai. Semua data dihapus secara langsung dari sistem kami (Firestore dan Firebase Authentication) dan tidak dapat dikembalikan.
